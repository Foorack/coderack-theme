# Coderack Theme

**Coderack is a Vim-insipred strong-contrast dark theme, for VSCode and code-server.**

Strongly recommended to use `catppuccin-latte` as your Icon theme.

```bash
ext install Catppuccin.catppuccin-vsc-icons
```
